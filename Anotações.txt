Linguagem de alto nivel - se aproxima a linguagem humana.
Utilizado pra criar scripts dinâmincos.
Um exemplo de um prédio:
O html é a estrutura, o css é o design e o JavaScript é a fiação e o encamento. 
É a parte interativa no desenvolvimento.

A tipagem funciona como um regra de dados, quanto mais forte mais obrigatório é a declaração do tipo de dado.
No JS é fraca então a declaração acontece de modo dinâmico.
Ex: se criar uma variável com valor entre aspas o JS já converte para string.

Tipos primitivos
boolean, null, undefined, number, string, array, object, function

variaveis são dados que variam, são como uma caixa onde pode-se guardar qualquer coisa.

para declarar:
var tem escopo global, pode ser alterada e local e se não definida é null

let tem escopo local de bloco, pode ser alterada e se não definida é null

const tem escopo local de bloco, somente leitura, o valor inciado é obrigatório e não pode ser alterado

escopo

escopo global a variavel é declarada fora de qualquer bloco, sua visibilidade fica disponivel para todo o código

escopo local a variável é declarada dentro de um bloco, sua visibilidade pode ficar disponivel ou não

regras de uso:

iniciar com letras, underscore_ ou cifrão, não iniciar com número
não iniciar com espaços(usar _)
não usar palavras reservadas como function, var, let, const
declarar as variáveis no topo do bloco do código

vetores ou arrays
lista ou matriz de variavel que possui indice
é um caixa que possui varias caixas dentro e cada um possui um valor
Ex: let array = ['string', 1, true...]

Array sempre entre [] e podem guardar qualquer valor dentro do indice
Ex; let array = ['srting', 1, true, false, ['array1'], ['array2']...]
o indice só é acessado por números inteiros, onde 0 é o primeiro e separados por virgula

manipulando arrays
depois de declarar o array tem vários métodos de manipulação

forEach() - itera um array;
push() - add item no final do array;
pop() - remove item no final do array;
shift() - remove item no início do array;
inshift() - add item no início do array;
indexOf() - retorna o índice de um valor;
splice() - remove ou substitui um item pelo indice
slice() - retorna uma parte de um array existente
